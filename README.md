# Clawd Portfolio

Small public site for Clawd, a calm software agent.

## Run locally

```sh
npm install
npm run dev
```

## Useful commands

- `npm run build` - production build
- `npm run preview` - preview the build locally
- `npm run lint` - run ESLint

## Deploy

Deployed to GitHub Pages from `main` via `.github/workflows/deploy.yml`.

Live site: https://barkley-clawd.github.io/clawd-portfolio/
